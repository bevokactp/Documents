== рецептор -- чувствительное нервное окончание или клетка, преобразующее воспринимаемое раздражение в нервные толчки

осязание -- ощущение, получаемое при прикосновении кожи к чему нибудь
хеморецепция -- восприятие веществ раздражителей из внешней или внутренней среды
ноцицепция == болевосприятие -- определение повреждение тканей
гигрорецепция -- восприятие влажность
осморецепция -- восприятие осмотическое давление
терморецепция -- восприятие теплоты
фоторецепция -- светочувствительное восприятие поглощённого света
электрорецепция -- восприятие электрические поля и электрические сигналы окружающей среды
механорецепция -- экстероцепция + проприоцепция + барорецепция + интероцепция
экстероцепция -- восприятие раздражений, воздействующих из окружающей среды
интероцепция -- восприятие раздражений, воздействующих из внутри
проприоцепция -- определение положение тела, напряжение и степень растяжения мышц и сухожилий
барорецепция -- определение давления
боль
вкус == смак -- вид хеморецепции ощущение, возникающее при действии веществ преимущественно на рецепторы вкуса
возбуждение == збудження -- ответ возникающий в каждой живой клетке или ткани любого организма как ответ на раздражение
гигрорецепция -- восприятие изменений влажности нервной тканью
головокружение == запаморочення -- ощущение мнимого вращения пространства вокруг себя или себя в пространстве (системное головокружение), либо ощущение чувства приближающейся потери сознания, неуверенности, шаткости (несистемное головокружение)
голод
зрение
кинестезия == проприоцепция -- ощущение положения частей собственного тела относительно друг друга и в пространстве
магниторецепция == биомагнетизм -- способность ощущать магнитное поле земли
обоняние == нюх -- ощущение запаха
осязание == прикосновение дотик -- ощущать воспринимать прикосновения и давление чульцами в коже, мышцах, слизистых оболочках
синестезия == соощущение -- возникновение нескольких ощущений при раздражении одного органа чувств
слух
тетрахроматия -- одновременное восприятие видимого диапазона спектра электромагнитного излучения четырьмя различными типами световых рецепторов
цветоощущение == цветовая чувствительность, цветовое восприятие, цветовое зрение -- воспринимать и преобразовывать световое излучение определённого спектрального состава в ощущение различных цветовых оттенков и тонов, формируя целостное ощущение
звуколокация -- эхолокация по звуковой волной
радиолокация -- эхолокация по лучистой волной
телепатия == мыслепередача
эхолокация == волноотбивание -- положение объекта определяется по времени задержки возвращений отражённой волны
запоминание == помин мнемоника мнемотехника запоминательство

импульс == толчок побуждение -- заставляющий начать или ускорить какую-либо деятельность
раздражимость == подразливість -- способность живого организма реагировать на внешнее воздействие окружающей среды изменением своих физико-химических и физиологических свойств
раздражитель == подразник стимул -- то, что воздействует на организм, вызывая в нём какую-либо ответку
физиология == теловеденье -- совокупность жизненных процессов, происходящих в организме и его органах
чувство -- способность воспринимать внешние впечатления, ощущать, испытывать что-нибудь
интуиция == наитие чутьё -- снисхождение, снисшествие чего-либо + знания из прошлых воплощений
ощущение == відчуття, чувственный опыт -- воспринимать органами чувств через раздражение чулец внешщними или внутреними подразниками
восприятие == перцепция -- обработка чувственных данных, включающая бессознательную и сознательную фильтрацию

тошнота == нудота -- тягостное ощущение в подложечной области, груди, полости рта и глотке, нередко предшествует рвоте
бредни == галлюцинация -- образ, возникающий в сознании без внешнего раздражителя
гусиная кожа == мурашки сироти -- небольшие пупырышки у основания мелкого волосяного покрова на коже,  когда холодно или же испытывает сильное возбуждение
