    нервная центральная система = цнс -- головный + спинной мозг
    нервная периферическая система == пнс -- из всех других нервов и нейронов, которые не лежат в пределах цнс
        нервная соматическая система == снс -- часть пнс, состоит из нервов, прикрепленных к коже, органов чувств и всех мышц скелета. отвечает за сознательные движения мышц и обработку сенсорной информации, поступающей через внешние раздражители: зрение, слух и осязание
    нервная вегетативная система == внс, автономная ганглионарная, органная, висцеральная, чревная -- часть пнс, отвечает за пищеварение, кровообращение, дыхание, обмен веществ
