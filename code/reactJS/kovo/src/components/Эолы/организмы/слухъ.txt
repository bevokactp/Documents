== звуквосприятие

ухо == слух вухо
ушная раковина -- внешняя часть уха млекопитающих
слуховые косточки -- костное образование в полости среднего уха наземных животных -- молоточек наковальня стремечко
турецкое седло -- образование в теле клиновидной кости черепа человека в виде углубления, напоминающего по форме седло
барабанная перепонка -- тонкая, непроницаемая для воздуха и жидкости мембрана, разделяющая наружное и среднее ухо
козелок -- небольшой хрящевой выступ на внешнем ухе, в передней части ушной раковины у млекопитающих
стремя == стремено -- одна из слуховых косточек среднего уха, соединяет наковальню и улитку (внутреннее ухо)
наковальня -- слуховая косточка среднего уха
ушная улитка -- это передний отдел перепончатого лабиринта
внутреннее ухо -- один из трёх отделов органа слуха и равновесия
наковальня == коваделко -- слуховая косточка среднего уха
молоточек -- слуховая косточка среднего уха
