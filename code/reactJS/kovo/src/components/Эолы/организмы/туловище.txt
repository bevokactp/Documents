== тулуп торс -- центральная в анатомическом отношении часть тела животных, исключая голову с шеей, конечности и хвост -- грудная клетка + живот + спина + таз

перси == груди грудь
брюшина == очеревина -- тонкая полупрозрачная серозная оболочка (пластинка), покрывающая внутренние стенки брюшной полости и поверхность многих внутренних органов
брыжейка == брижа оточина -- член посредством которого полые органы брюшной полости прикреплены к задней стенке живота
брюховица == брюхо, желудок
чрево == живот брюхо, пузо, утроба, нутро кишеня
пуп -- рубец на передней брюшной стенке, остающийся после отпадения остатков пуповины у новорождённого ребёнка
висцеральный == внутренностный -- относящийся к внутренним органам животного
киль -- виріст кістки груднини у хребетних тварин, слугує для додаткового прикріплення сильно розвинутих грудних м'язів
спина -- задняя сторона туловища, простирающаяся от нижней части шеи до поясницы
мякоть -- мягкая мышечная часть тела животных, человека
талия == поперекъ, поясъ -- наиболее узкой частью туловища

кульша -- в позвоночных место соединения нижней конечности с туловищем. часть бедра образуют три кости таза , которые у взрослых срастаются в одну тазовую кость (что также называется вертлужной)
промежность -- в широком смысле слова -- комплекс мягких образований, расположенных между лобковыми костями спереди, седалищными буграми седалищных костей с боков и верхушкой копчика сзади
лонное сочленение == лобковый симфиз -- расположенное по срединной линии вертикальное соединение верхних ветвей лобковых костей
стегно == чресло бедро пах клуб кульша -- часть нижней конечности человека (задней конечности у животных) между тазобедренным и коленным суставами
пах == пахвина -- часть нижнего края брюшной области, примыкающая к бедру
поясница -- нижняя часть спинного хребта
седалище == ягодицы зад дупа срака жопа задница гузка
гузка -- хвостовая клинообразная часть туловища птицы

телосложение == комплексия
