== костяк скелет склет

хорда = тяж струна -- длинный эластичный продольный у хордовых как осевой скелет
лобковая кость -- у позвоночных животных парная кость, одна из трёх, образующих при сращении тазовую кость
седалищная кость -- у позвоночных животных парная кость, одна из трёх, образующих при сращении тазовую кость
тазовая кость -- это обозначение для парной кости правой и левой, которые относятся к поясу нижних конечностей, а вместе с крестцом и копчиком образуют таз
ребро -- одна из парных дугообразных плоских костей, идущих от позвоночника к грудине составляя грудную клетку у позвоночных
груднина == грудная кость --  у многих наземных позвоночных. даёт опору плечевому поясу. у земноводных и рептилий грудина обычно хрящевая, у птиц костная
крыльца == лопатки
лопатка -- кость пояса верхних конечностей, обеспечивающая сочленение плечевой кости с ключицей
ключица -- трубчатая кость s-образной формы из пояса верхней конечности, соединяющая лопатку с грудиной
таранная кость -- одна из костей предплюсны, формирующая нижнюю часть голеностопного сустава посредством соединения с больше- и малоберцовой костями
таз -- часть туловища, пояс скелета нижней конечности, состоящий из двух тазовых костей и крестцовой , которые крепко соединяются между собой и составляют прочную опору для свободных нижних конечностей
сошник == леміш -- непарная кость, в виде пластинки трапециевидной формы, в носовой полости в сагиттальной плоскости и вместе с перпендикулярной пластинкой решётчатой кости образует костную перегородку носа
бедренная кость  -- единственная кость бедра. самая большая и длинная трубчатая кость в организме человека. как и все длинные трубчатые кости, имеет тело и два конца
носовая перегородка == перетинка -- вертикальная пластинка, разделяющая носовую полость на две половины
груть -- одна из частей туловища. образуется грудиной, 12 парами рёбер, грудным отделом позвоночника, мышцами

позвоночник == хребет, становая жила, позвоночный хребетний стовп -- несущий элемент скелета позвоночных животных
    32-33 позвонков
        позвонок == хребец -- костные или хрящевые составляющие позвоночника позвоночных животных
        7 шейных
            атлант -- первый шейный позвонок позвоночных животных
        12 грудных
        5 поясничных
        5 крестцовых, соединенных в крестец
            крестец == крижова кістка, крижі -- часть позвоночника наземных позвоночных, обеспечивающая его соединение с тазом между поясничными и хвостовыми (у человека -- копчиковыми) позвонками
        3 - 4 копчиковых
            копчик == хвостик куприк, копчиковая куприкова кістка -- нижний отдел позвоночника из 4-5 сросшихся позвонков
    между которыми расположены 23 межпозвоночных диска.
        межпозвоночный диск -- эластичное волокнистохряжевое соединение между позвонками

цевка -- одна из костей ноги птиц, расположенная между голенью и пальцами
раковина моллюсков -- наружное скелетное образование, покрывающее тело большинства моллюсков

мослы == кости
кость == кистка --  твёрдый орган преимущественно из костной ткани позвоночных животных
связка -- плотное образование из соединительной ткани, скрепляющее части скелета или внутренние органы
палый == имеющий столько-то пальцев
безпалый == утративший один или несколько пальцев
хрящ -- вид соединительной ткани, отличается плотным, упругим межклеточным веществом, образующим вокруг клеток-хондроцитов и групп их особые оболочки, капсулы
