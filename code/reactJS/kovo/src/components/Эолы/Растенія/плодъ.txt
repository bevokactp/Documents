
-- конечное видоизмение цветка покрытосеменных растений для образования, защиты и распространения заключённых в нём семян


качан

ложный -- в образовании принимают участие и другие члены (цветоложе, околоцветник) -- яблоня груша земляника
настоящий == истинный -- образован из разросшейся завязи
    простый -- образован из одного пестика
        монокарпные -- пестик образован одним плодолистиком
        ценокарпии --  несколькими сросшимися плодолистиками
    сборный == сложный -- образован из многочленного плодолистника цветка, состоящим из нескольких свободных пестиков, каждый из которых образован одним плодолистиком
    дробный -- из цветка с многогнёздным членом из нескольких сросшихся плодолистиков. после созревания распадаются на части, по числу гнёзд завязи (числу плодолистиков)
бессеменной -- не содержат семян
партенокарпический -- образованный без оплодотворения (обычно бессеменные)
сухой -- с сухим околоплодником
    коробочковидные -- многосемянные
		коробочка -- лопающийся или растрескивающийся, из двух и более плодолистиков (мак, тюльпан, дурман)
		крыночка -- лопающийся или растрескивающийся (очный цвет, подорожник, белена чёрная)
		бобы (зернобобовые и бобовые)
		мешочек -- одно/немного -семянный, образован сросшимися краями прицветника и тонким пленчатым околоплодником (осока)
		стручок -- из двух плодолистиков с двустворчатой оболочкой и внутренней перепонкой (крестоцветные)
		стручочек -- двугнёздая коробочка, образован двумя плодолистиками и вскрывающаяся двумя створками, по брюшному и спинному шву (крестоцветные)
		листовка -- многосемянка, вскрывающимся по брюшному шву (орлики лютиковых живокость калужница банксия)
		вислоплодник -- сухой двусемянный дробный (зонтичные аралиевые)
    ореховидные == односемянные
		орех -- со съедобным ядром и твёрдой скорлупой (лещина, фундук, лютик)
		орешек -- невскрывающийся односемянный мелкий с деревянистым околоплодником (гречиха клубника конопля щавель острица шиповник)
		зерновка -- односемянный нераскрывающийся (злаки)
		крылатка -- орех с крылом (клён гречишные берёзовые)
		жёлудь == жолудь -- односемянный синкарпный нижний плод с жестким кожистым околоплодником, частично или полностью заключённый в плюске (дуб буковые)
		семянка -- односемянный невскрывающийся плод с кожистым околоплодником (подсолнечник)
		костянки сухая с кожистым межплодником (грецкий орех)
сочный -- с мясистым и сочным околоплодником
    ягодовидные -- многосемянные
        ягода -- из единой завязи (плод черники, смородины, томата)
		яблоко -- многосемянный нераскрывающийся (яблоневые, груши, рябины)
		тыквина (тыквенные)
		померанец == гесперидий -- нераскрывающийся многогнёздый и многосемянный плод, близкий к ягоде, околоплодник состоит из толстой и мягкой кожи, покрытой цедрой (цитрусовые)
		гранатина -- плод, характерен для представителей рода гранат
    костянковидные
        костянка сочная -- разделениеслоёв околоплодника: тонким кожистым внеплодником, мясистым межплодником и одревесневшим внутриплодником, заключающим семя и образующим твёрдую косточку -- (вишня, слива, персики калина, пальма кокосовая , оливы)
	соплодие == супліддя -- совокупность сросшихся или тесно сближенных плодов, либо любая совокупность плодов, образующаяся из соцветия
	реп'ях -- суцвіття і плоди, що мають численні колючки і здатні чіплятися до одягу людей, хутра тварин
	многоорешек -- орешковый плод, состоящий из обычно односемянных орешков, окружённых твёрдым невскрывающимся околоплодником


косточка -- покрытое деревянистой оболочкой ядро
межплодник == мезокарпий -- часть плода растения, между эндокарпием и экзокарпием
плюска == мисочка -- чашевидный орган, окружающий плод некоторых растений порядка букоцветные
сикóний -- полое округлое образование у растений рода фикус, к которым относится, в частности, инжир
фруктъ == плодъ древоплодъ
цедра == корка
мякоть -- мягкая подкожная часть плодов, корнеплодов, зерна
альбедо -- внутренний белый рыхлый губчатый слой кожуры плодов цитрусовых
околоплодник == опліддя навколопліддя оплодень перикарпий -- часть плода семенных растений. оболочка, окружающая семена и формирующая внешний вид и форму плода, обычно составляет основную часть плода и служит для защиты семян от внешних воздействий
семена == насіння -- для размножения и расселения семенных растений, развивающаяся после оплодотворения из семязачатка и содержащая зародыш
семязачаток == семяпочка -- образование у семенных растений, из которого развивается семя
плодоножка -- стеблевой орган, несущий плод из цветоножки
